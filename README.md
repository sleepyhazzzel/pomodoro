## 🍅 Pomodoro 番茄工作法定時器
### How to use
- 可播放、暫停、跳至下一個項目
- 可在「事項」 新增、編輯、調整順序、刪除預定事項
- 「設定」會彈出視窗可開關、更換鈴聲
